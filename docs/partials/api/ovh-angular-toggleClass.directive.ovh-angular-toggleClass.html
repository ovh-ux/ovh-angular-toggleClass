ovh-angular-toggleClass
directive in module ovh-angular-toggleClass

Description

To use ovh-angular-toggleClass, you had to inject ovh-angular-toggleClass as dependency and use it like this example inside a controller 

 angular.module("myApp", ['ovh-angular-toggleClass]);

Then, you can use the directive in your HTML code

Usage

as element:
<ovh-angular-toggle-class
       toggle-class-group="{string}"
       toggle-class-side="{bool}"
       toggle-class-target="{string}"
       toggle-class-target-item="{string}"
       [toggle-class-target-backdrop="{string}"]
       [toggle-class-name="{string}"]
       [toggle-class-trigger-close="{string}"]
       toggle-class-trigger-open-close-event="{function}"
       [toggle-class-click="{string}"]>
</ovh-angular-toggle-class>
as attribute
<ANY ovh-angular-toggle-class
     toggle-class-group="{string}"
     toggle-class-side="{bool}"
     toggle-class-target="{string}"
     toggle-class-target-item="{string}"
     [toggle-class-target-backdrop="{string}"]
     [toggle-class-name="{string}"]
     [toggle-class-trigger-close="{string}"]
     toggle-class-trigger-open-close-event="{function}"
     [toggle-class-click="{string}"]>
   ...
</ANY>
